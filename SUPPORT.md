# Support

Thank you for your interest in LedgerForge.

If you need help, please use one of the following channels.

## Questions

For general questions, please use:

- GitHub Discussions

Please avoid opening Issues for usage questions.

---

## Bug Reports

If you found a bug:

- Search existing Issues first.
- Open a new Issue if it has not already been reported.
- Include steps to reproduce.
- Include logs when possible.
- Never include sensitive financial information.

---

## Feature Requests

Feature requests are welcome.

Please explain:

- the problem you're trying to solve
- your expected workflow
- why the feature would benefit other users

---

## Security Issues

Please **do not** report security vulnerabilities publicly.

See the SECURITY.md document for responsible disclosure instructions.

---

## Commercial Support

Commercial support and consulting may be available in the future.

Please refer to the project website when available.

---

Thank you for helping improve LedgerForge.
